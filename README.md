```
「幸せを築く人を、幸せに。」これが私たちの掲げるミッションです。

本来、ものづくりに携わる人々は、誰かに幸せを届ける人たちです。

そんな方々がもっとクリエイティブに、もっと豊かに働けるよう、
私たちは熱い想いで日々現場に向き合っています。
```
